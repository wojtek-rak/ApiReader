# ApiReader
App for reading and displaying API from Github.
Enter Username and Repositoryname and click Get Data.
You can refresh it in time by enter interval time ( in sec )

NUnit <br />
MoQ <br />
System.Reactive <br />
Akka <br />
Akka.TestKit <br />
